# Super Admin Access

## Direct Link to Super Admin Dashboard

When running on **port 3000**, access the super admin dashboard at:

```
http://localhost:3000/?admin=dukaAdmin
```

Or simply:
```
http://localhost:3000/?admin=dukaAdmin
```

## Features Available

The Super Admin Dashboard includes:
- Manage subscription plans
- View all store subscriptions
- Configure payment methods
- Monitor payment history
- Manage billing settings
- View transaction logs
- Configure SMS reminders
- Manage promotional offers

## Starting the Application

```bash
npm run dev
```

Then open:
```
http://localhost:3000/?admin=dukaAdmin
```

## Production Deployment

For production deployments, the super admin link would be:
```
https://yourdomain.com/?admin=dukaAdmin
```

Make sure to keep this link secure and share only with authorized administrators.
